# Nutrición

Documentación nutricional personalizada para una fase agresiva de pérdida de grasa de 14 días.

## Archivos

- `encuesta-preferencias.md`: encuesta de preferencias alimentarias y respuestas del usuario.
- `plan-nutricional-14-dias.md`: estrategia nutricional rotativa basada en las preferencias recogidas.

## Objetivo de referencia

| Variable | Objetivo aproximado |
|---|---:|
| Calorías | 1.820 kcal/día |
| Proteína | 190 g/día |
| Grasas | 55 g/día |
| Hidratos | 141 g/día |
| Duración | 14 días |

La dieta prioriza adherencia, alimentos fáciles de conseguir en supermercados españoles y reparto de hidratos alrededor del entrenamiento.
