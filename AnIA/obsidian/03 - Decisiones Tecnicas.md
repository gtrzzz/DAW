# Decisiones Tecnicas

Fecha: 30/05/2026

## 30/05/2026 - Usar PC principal para Ollama

Decision:
El PC principal ejecutara Ollama y los modelos de IA.

Motivo:
Tiene GPU NVIDIA RTX 5070 con 12 GB de VRAM, 32 GB de RAM y CPU Ryzen 5 7600X.

Consecuencia:
Si el PC esta apagado, la IA no respondera salvo que se implemente Wake-on-LAN o un modo alternativo.

## 30/05/2026 - Usar portatil como servidor

Decision:
El portatil ejecutara Telegram, web, backend y servicios 24/7.

Motivo:
Consume menos y puede estar siempre encendido.
