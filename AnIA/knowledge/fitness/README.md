# Fitness

Guarda aqui documentos sobre fitness, nutricion, hipertrofia y deporte.

Estructura recomendada:

```text
nivel-1/        Fuentes cientificas y bases de evidencia.
nivel-2/        Divulgacion basada en evidencia.
nivel-3/        YouTube fiable y transcripciones revisadas.
notas-propias/  Resumenes, protocolos y decisiones personales.
```

Usa `plantilla-documento-rag.md` para crear documentos con fuente, URL, nivel, categoria, limitaciones y aplicacion practica.
