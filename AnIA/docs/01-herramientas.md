# Herramientas Del Proyecto

Fecha: 30/05/2026

## Herramientas Instaladas O Recomendadas

- Git: control de versiones.
- GitHub CLI: conexion con GitHub desde consola.
- VS Code: editor de codigo.
- Obsidian: documentacion personal y notas tecnicas.
- Ollama: IA local en el PC principal.

## Uso Recomendado

GitHub guarda el codigo y la documentacion final. Obsidian sirve para pensar, planificar, guardar decisiones y recopilar problemas encontrados.

## Flujo De Trabajo

```text
Idea -> nota en Obsidian -> tarea en GitHub -> cambio en codigo/docs -> commit -> push
```

## Fechas

Usar siempre formato `dia/mes/ano`, por ejemplo `30/05/2026`.
